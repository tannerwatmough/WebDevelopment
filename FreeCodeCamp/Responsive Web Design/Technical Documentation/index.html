Python Introduction

Python is an easy to use programming language that is frequently used in data science and scientific disciplines. It is also popular among Computer Scientists in the Machine Learning field.

Python does abstract some important computer science topics from the user such as memory allocation. It is also a interpreted language which means it can run slower than programs that are compiled like C and C++.

Conditionals

Conditionals are if statements that will only execute the program if certain conditions, hence conditionals, are met.

if x < 3 do:
print("Hello")

In this case, the code would only run if the variable x was less than 3. If x was less than 3, the console would print out the statement "Hello".

Loops

Loops will continue executing sequentially but loop around at the end of the loop until the loop has a break/end command or meets a conditional.

  • while True:
  • This is a common way to form a pseudo-infinite loop to keep a program running until the program is exitted or terminated.
  • Loops can contain conditionals that might lead to the termination of the loop or program.
while True do:
if x > 3 do:
x = x - 1
else:
exit

In this case, the loop would continue until x < 3, at which point it would exit the loop and end the program.
